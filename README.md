# Stock-Price-EDA
The aim here is to be able to do the following:
  -> Be able to pull data with yfinance
  -> Basic recognition of patterns and ability to analyse trends
  -> Present volatility, moving averages 
and obviously visualising using plotly.

Beyond the initial scope further goals (which I might end up making another repository of instead) include but are not limited to:
  -> Accessibility to multiple API's
  -> Correlating Reddit sentiment with market trends
  ->better visualisations
